# Data formats in Earth Sciences: CSV

CSV files are often used in earth sciences and have many more implications. CSV stands for comma-separated values. A CSV file is used to store data recoreds and it can be used in a variety of programs, like spreadsheet programs, statistical or mathematical programs. 

The characteristics of a CSV files are:
- data is separated by comma or (semi)colon
- each record is one line
- plain text format file
- used for information exchange, because it can be used by lots of different tools

### History
The CSV file has a long history. It was already used in 1972 by IBM Fortran programming language. Fortran was coded using punched carts and CSV formats were the best format to use.
CSV files are so basic using there plain text format, that they are widely used for exchanging information.
